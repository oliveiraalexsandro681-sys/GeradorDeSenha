# Gerador de Senhas (WinForms - C#)

Projeto desenvolvido em C# utilizando Windows Forms.

🚀 Funcionalidades

* Cadastro de nome e e-mail
* Validação de campos obrigatórios
* Geração automática de senha aleatória
* Simulação de processamento utilizando `Task.Delay()`
* Exibição da senha gerada ao usuário

🛠️ Tecnologias

* C#
* Windows Forms (.NET)

📚 Objetivo

Projeto desenvolvido para prática de programação em C#, focando em lógica de programação, validação de dados, geração de números aleatórios e programação assíncrona.
